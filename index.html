üìã Registry Structure

The registry follows the MCP Server Schema specification and includes:

  • Server metadata (name, description, version)
  • Repository information
  • Package distributions (npm, Docker, PyPI, NuGet, MCPB)
  • Environment variables and configuration
  • Transport protocols
